Principais Formatações

Negrito

Para deixar uma palavra de destaque em seu texto basta usar a tag strong

Italico

Para deixar uma frase ou palavra em italico deve ser usado a tag em desse jeito aqui

Texto pequeno

Para deixar o texto pequeno para contratos por exemplo a tag usada ser a small assim: Lorem ipsum, dolor sit amet consectetur adipisicing elit. Ipsa ratione accusamus provident accusantium veritatis vel, ea aliquid sequi perspiciatis nemo sit eligendi? Quod laboriosam fugit voluptates iste vitae perspiciatis commodi.

Texto sublinhado ou deletado

Existe a também como deixa seu texto ou frase sublinhado utilizando a tag ins assim: you belong with me e deletado utilizando a tag del assim:Taylor Swift

Outras

Para deixar um numero elevado deve ser utilizado a tag sup desse jeito: 20x2 e para deixar um numero abaixo é utilizado a tag sub assim: H2O
